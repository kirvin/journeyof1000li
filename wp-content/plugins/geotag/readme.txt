Provides geocoding features for Wordpress.

== Description ==

With this plugin you can add coordinates to your posts and display maps with these positions. On these maps you can also display KML/KMZ files or the geotags of the images of your post. The plugin fits into the Wordpress admin panel, is easy to use and can be individually configured to suit your needs.

= Features =

* **Set geographic coordinates** for every post or page.
* Add one or more **interactive maps** to your posts or pages.
* Display **KML/KMZ files** with your route, your waypoints or your photos.
* Read the geotags from the **exif header** of the photos of your post and display them.
* Can be configured individually for every post.
* Perfect for **blogging via email** as you may set the coordinates and options in the post's text.
* One-file-design makes it easy to costumize the plugin.
* Pic up your location by a click on the map or by searching for an address
* Costumize your map style.
* Supports **querys to display multiple posts** in one map.
* Supports static maps for higher speed.
* **Adds maps automatically** or by using a shortcode within the text of the post.
* Supports **different geographic coordinate conversions**.
* Add geotags to your **feeds or HTML header**.
* Detailed documentation.
* Inspired by and compatible with [Ben Huson's WP Geo](http://wordpress.org/extend/plugins/wp-geo/).

== Installation ==

1. Unzip the package. Perhaps you want to delete the screenshots to save some disk space.
1. Upload the folder `geotag` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Set the Google Map API key at the configuration page.
1. If you have updated the plugin, check the configuration and click save. This is important as the structure in which the settings are saved may have changed.
